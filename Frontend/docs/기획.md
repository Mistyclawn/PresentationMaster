구체적 구상
1. 호환성
PPTX 파일을 편집 가능해야 하는가? -> 꼭 그럴필요는 없지만 가능하면 좋다.
프로그램에서 -> MS PPT로 일방적 출력만 가능해도 충분?

2. 발표 설계 기능
유저가 임의로 만든 발표 서류 - 스크립트 등 양식을 지원하되 기본은 프로그램 내부에서.
스크립트 작성 화면과 슬라이드 작성 화면을 구분하자.

3. 그래픽 디자인
여러 테마와 그래픽 요소를 지원해야 한다. 최소한 Canva, MS PPT에서 지원하는 기본 그래픽 요소(원, 사각형, 자동 텍스트 레이아웃, 텍스트 박스, 이미지 등)를 지원해야 한다.

4. 사용환경
오프라인 환경에서 사용할 예정, 온라인도 가능하면 좋음.-> 따라서 LLM API, 로컬 모델 등 옵션 제공해야 함
